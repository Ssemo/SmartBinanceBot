# SmartBinanceBot 🔥

بوت تداول ذكي على منصة Binance (Spot و Futures) مدعوم بالذكاء الاصطناعي.

### وظائف البوت الرئيسية:
- **تحليل ذكي**: يجمع البيانات من عدة مصادر ويحلل السوق باستخدام الذكاء الاصطناعي
- **استراتيجيات تداول متقدمة**: DCA, Grid, Trend Following, وإستراتيجيات مخصصة
- **تداول آلي ذكي**: يتخذ قرارات تداول قريبة جداً من أفضل البوتات الموجودة حالياً
- **قابل للتعلم**: يتعلم من السوق ومن قراراته السابقة ويتطور مع الوقت
- **إدارة مخاطر محترفة**: Stop Loss, Take Profit, Position Sizing ذكي

### الهدف الرئيسي:
زيادة رصيد التداول بشكل مستمر مع الحفاظ على نسبة مخاطر منخفضة.

---

### خطوات التشغيل:
1. `git clone https://github.com/Ssemo/SmartBinanceBot.git`
2. `cd SmartBinanceBot`
3. `pip install -r requirements.txt`
4. أنشئ ملف `.env` وضع فيه مفاتيح Binance API
5. `python main.py`

**تحذير هام**: التداول يحمل مخاطر عالية. جرب البوت أولاً على حساب تجريبي.

# 🤖 SmartBinanceBot
> بوت تداول ذكي لـ Binance (Spot & Futures) مدعوم بالتحليل الفني.

## 🌟 الميزات
- 🔍 **تحليل ذكي**: يجمع البيانات ويحلل السوق تلقائياً.
- 📊 **استراتيجيات متعددة**: يدعم DCA، Grid، واتجاهات السوق.
- 🛡️ **إدارة مخاطر**: نظام Stop Loss و Take Profit متطور.
-  **تعلم ذاتي**: يحسن قراراته بناءً على البيانات السابقة.

## 📦 التثبيت (Installation)

1. قم بتحميل المشروع:
```bash
   git clone https://github.com/Ssemo/SmartBinanceBot.git
   cd SmartBinanceBot
```

2. ثبّت المكتبات المطلوبة:
```bash
   pip install -r requirements.txt
```

## 🔑 الإعدادات (Configuration)

1. أنشئ ملف `.env` وضع فيه مفاتيحك:
```env
   BINANCE_API_KEY=your_api_key
   BINANCE_SECRET_KEY=your_secret_key
```
   ⚠️ **تنبيه أمني**: لا تشارك مفاتيحك مع أي شخص، ولا ترفع ملف `.env` للعامة.

## 🚀 التشغيل (Usage)

لتشغيل البوت وتشغيل التحليل، نفذ الأمر التالي:
```bash
python3 main.py
```
